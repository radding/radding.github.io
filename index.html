Projects & Experience

Barracuda Networks - Intern

Worked on the cloud appliance for the mail archiver. I built a support tool to allow Support to look up and verify customer info. I upgraded the libraries and dependencies used by the entire systems. This included building modules that mimicked behaviour that was no longer supported. Also made various bug fixes and tools.

LykeMe - Founder

A social media site that matches people in your area based on shared intrests. Designed and built the backend architecture and code to support the app. Led the development of the front end as well as the back end. Built in iOS and python

Card Tracker

A web crawler that searches the web for pictures with credit card numbers in them, and then alerts the respective issuers to notify that the number was found online. Uses OCR built using openCV and tesseract. Built using C++ and Python

Pigmnt

An iPhone App that tracks colors using the camera and in real time. Can be used with the google cardboard for augmented vision. It uses openCV to find the colors, trace the outline and build a box around the outline. Built in C++, Objective-C, and Objective-C++.

Turnd-On

A social media application that is based soley on text messaging. A user subscribes to a "channel" and will recieve text messages when some one posts (texts) to this group. Built using Python.

Space odyssey

A Virtual Reality game built using the Unity game engine, supporting network game play. It uses the occulus rift and google card board for the graphics and a myo guesture recognizer as a controller. Built using C# and Java.

HIT web team - Web Developer

Built and mainted web sites at MSU Health IT. Set up proper testing enviroments and designed the modules for the sites. Used technologies such as C#, XSL/XML, HTML, CSS, JavaScript, jQuery, and PHP.

Battle Bots

A Robot that was built using Raspberry Pis and arduinos that supported Streaming of video over a network and controllable using the MYO guesture controller. Built using C, Python and Java.

Spartahack Website

Built the website for Spartahack 1. It included a Content management system. Also set text messages to mentors and allowed us to make push notifications to iOS devices. Built using Python (django).
